### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（客户）、律师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 律师管理
6. 案源信息管理
7. 案件信息管理
8. 结案评论管理
9. 事务所意见管理
10. 律师意见管理
11. 公告信息管理

**律师功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 案源信息管理
6. 案件信息管理
7. 结案评论管理
8. 律师意见管理
9. 公告信息管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 律师管理
5. 客户管理
6. 案源信息管理
7. 案件信息管理
8. 结案评论管理
9. 事务所意见管理
10. 律师意见管理
11. 基础数据管理（案源类型管理、公告类型管理）
12. 公告信息管理
